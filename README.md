# Analise-Dados-SQL

# 📊 Portfólio de Análise de Dados SQL

Este repositório contém projetos de estruturação e análise de bancos de dados relacionais utilizando **DBeaver** e **SQLite**.

## 🎮 Projeto: Vendas de Jogos PS4
Neste projeto, realizei a normalização de uma base bruta (CSV) para um modelo relacional.
- **Tecnologias:** SQLite, DBeaver.
- **Habilidades demonstradas:** Normalização, Joins, Window Functions (`RANK`).
- **Insight Principal:** Identifiquei que o gênero 'Action' domina o mercado americano, enquanto 'Role-Playing' tem maior retenção no Japão.

## 🛒 Projeto: Superstore Analytics
Foco em KPIs empresariais e performance de vendas.
- **Consultas principais:** Faturamento anual, Ticket médio e Margem de Lucro.
